What is Cloud Backup?

Delivered on Microsoft Azure, Innovative V's Cloud Backup service consists of two core solutions, relevant depending on the type of environment and workload to be protected - in the cloud or on-premises.

Cloud Backup for Azure Workloads

The Azure Cloud Backup service, integrated into Azure Virtual Machines (Azure VMs), is cost effective, can be instantly enabled and is simple to use. It can be switched on for both new and existing deployments of Azure VMs and the whole VM and associated data is protected.

Cloud Backup for On-Premises Workloads

Cloud Backup can protect on-premises physical servers and VMs running on Hyper-V and VMware. It can either integrate with an existing System Center Data Protection Manager platform or be deployed as a new Microsoft Azure Backup Server to provide a disk-to-disk-to-cloud solution.
